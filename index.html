• Infinite Flight Airport Editing

Glad to have you here! If you enjoy aviation, designing, and want to make Infinite Flight better, then you've come to the right place! Infinite Flight contains many airports worldwide, and our mission is to make sure they are as accurate as possible. We're looking for people to help with creating accurate layouts, with taxiways, taxi lines, ATC frequencies and parking spots.

Getting Started

Getting started is very easy. The video above will give you a full, guided, step-by-step guide on how to prepare for editing airports. Here are the steps listed:

1. Create a GitHub account, by entering your details on the GitHub website. Remember these details, they'll come in handly later!

2. Direct message one of the airport editing admins on Slack your GitHub username. They'll invite you to the GitHub team. Accept that invite on the GitHub website or in your email. (If you're not a part of the Slack, scroll down to the Slack section to get more information on how to get invited.)

3. Download the GitHub Desktop app.

4. Install a copy of X-Plane. Skip this step if you already have a DVD/Steam copy, otherwise you can download the free demo from here.

5. Install a copy of World Editor from here.

6. Login to the GitHub Desktop app, then tap the "+ button on the top left corner, select the "Clone" tab, and select "Airports". Save this to somewhere you'll remember later.

7. Open WorldEditor, and select the folder in which you installed X-Plane to. Create a new scenery package, and call it the name of the airport you want to update.

8. A new window should appear. Go to File - Import from apt.dat.

9. Find the folder in which you cloned the repo to earlier. Select the airport you want to edit, and import the apt.dat.

10. Download Java (skip this if you already have it installed) and WEDBing, then run WEDBing and add the path to your X-Plane folder in the top left.

Airport Editing Checklist

Do's and Don'ts

Do's

  • Use accurate imagery for airports.
  • Add taxiways.
  • Add taxiway lines.
  • Add parking spots.
  • Modify runways.
  • Add airport boundaries.
  • Snap vertices.
  • Cut holes for overlapping taxiways.

Don'ts

  • Add runway markings with taxiway lines (this includes runway numbers, blastpad arrows, etc.).
  • Forget to include a boundary.
  • Make up an airport - it HAS to be a real airport.

Sending Changes

  • In WorldEditor, go to File - Export as apt.dat.
  • Save the file in the corresponding folder for the airport.
  • Open up an email client (such as Gmail).
  • Send an email to airports@liveflightapp.com.
